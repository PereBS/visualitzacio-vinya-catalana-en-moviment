# Datasets processats

Aquesta carpeta conté els datasets processats principals utilitzats com a base analítica del projecte.

Aquests fitxers no són necessàriament els que carrega directament Observable, sinó les versions intermèdies o finals que permeten entendre i reproduir la preparació de dades del projecte.

## Fitxers previstos

```text
panell_strict.csv
  Panell comarcal-anual amb dades de superfície, producció, rendiment i clima directe quan existeix cobertura comarca-any.

panell_proxy.csv
  Versió del panell amb reconstrucció proxy de les variables tèrmiques per a comarques sense cobertura directa completa.

meteo_comarca_any.csv
  Dades meteorològiques agregades per comarca i any, utilitzades per construir indicadors climàtics anuals i estacionals.

ervc_agregat_varietat_comarca_2008_2024.csv
  Versió agregada del Registre Vitivinícola de Catalunya per a l'anàlisi varietal i territorial.
